Listas

La lista es la estructura mas vacica de Python, son mutables es decir no es necesario definirlo con un tamaño fijo, y se puede modificar la informacion, como adicionar, eliminar y modificar con los comandos (append, remove o pop ) 

            familia=["rodney","cecilia","juan miguel","samuel"]
            print(familia)

            familia[3]="Samuel david"
            print(familia)

            familia.append("elkin jose")
            print(familia)
se puede adicianar al final de la lista usando append o especifiando la posicion [-1] ejemplo: 
            familia.append("elkin jose")
            print(familia)        

            familia[-1]="angelica"
            print(familia)
remover valores de una lista 
            familia.remove("rodney")
            print(familia)

            
            familia.pop()
            print(familia)
Tambien podemos ordenar la lista con el metodo sort 
            #ordenamos
            familia.sort()
            print(familia)
ahora vamos a recorrer la lista a traves de un for 
            #recorrer la lista a travez de un ciclo for
            print("---ciclor for de un alista")
            for nombre in familia:
                print(nombre)
Copiar lista, por ejemplo de la lista familia a familia2 
            #copiar familia
            print("----creando copia de lista -----")
            familia2=familia.copy()
            print(familia2)
sublista, podemos escoger dentro de una lista un rango especifico por ejemplo : 
            #sublisa
            print("----creando copia de lista -----")
            subfamilia=familia[1:3]
            print(subfamilia)
Ejercicio de agregar una lista frutas y si ya existe una determinada fruta eliminarla despues ver el resultado y recorrer la lista con un ciclo for 
            listFrutas=["uva","pera","manzana","banana"]
            print(listFrutas)
            listFrutas.append("naranja")
            print(listFrutas)

            #condicinarl
            if "pera" in listFrutas:
                listFrutas.remove("pera")

            print(listFrutas)
            print("--ciclo for ----")
            for fruta in listFrutas:
                print(fruta)

FastAPI

FastAPi nos ayuda a crear una API Rest con Python, los pasos para trabajar con eta libreria seria:
  • Creamos un entorno virtual de Desarrollo
  • Activar entorno virtual de Desarrollo
  • Instalar Dependencias en entorno virtual de Desarrollo
  • creamos el proyecto main.py

Creamos entorno virtual

Desde la terminal que se encuentra dentro del Visual studio escribimos: 
            python -m venv venv
Esto crea una carpeta venv que es donde instalaremos todas las dependencias de nuestro proyecto.
Ahora debemos activar nuestra carpeta virtual: 
            en windoes hay que abrir cmd y escribir:
            ven\cripts\activate
        
            en linux desde la termila hay que escribir:
             source venv/bin/activate
ahora instalamos FastAPI en el entorno virtual activo desde Cmd escribimos: 
             pip install fastapi uvicorn
cuando instale el fastapi me salio que debia actualizar el pip de la siguiente forma: 
            python -m pip install --upgrade pip
nota:
no me dejaba instalar el entorno desde la terminal de visual studio me todo escribir lo siguiente: 
            Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
Ejecutamos el proyecto escribiendo : 
            uvicorn main:app
si queremos que el servicio se mantenga abierta debemos escribir : 
            uvicorn main:app --reload
Instalamos el driver de posgretSql de la siguiente forma: 
            pip install psycopg
La instalacion del psycopg no se instala correctamente con esta verion, toco instalarlo de la siguiente forma: 
            pip install "psycopg[binary]"
El driver para sqlexpress 2022 debemos inatalar lo siguiente: 
            pip install pyodbc
para verificar si tego la version instalada se escribe: 
            psql --version
