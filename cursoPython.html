Listas

La lista es la estructura mas vacica de Python, son mutables es decir no es necesario definirlo con un tamaño fijo, y se puede modificar la informacion, como adicionar, eliminar y modificar con los comandos (append, remove o pop ) 

            familia=["rodney","cecilia","juan miguel","samuel"]
            print(familia)

            familia[3]="Samuel david"
            print(familia)

            familia.append("elkin jose")
            print(familia)
se puede adicianar al final de la lista usando append o especifiando la posicion [-1] ejemplo: 
            familia.append("elkin jose")
            print(familia)        

            familia[-1]="angelica"
            print(familia)
remover valores de una lista 
            familia.remove("rodney")
            print(familia)

            
            familia.pop()
            print(familia)
Tambien podemos ordenar la lista con el metodo sort 
            #ordenamos
            familia.sort()
            print(familia)
ahora vamos a recorrer la lista a traves de un for 
            #recorrer la lista a travez de un ciclo for
            print("---ciclor for de un alista")
            for nombre in familia:
                print(nombre)
Copiar lista, por ejemplo de la lista familia a familia2 
            #copiar familia
            print("----creando copia de lista -----")
            familia2=familia.copy()
            print(familia2)
sublista, podemos escoger dentro de una lista un rango especifico por ejemplo : 
            #sublisa
            print("----creando copia de lista -----")
            subfamilia=familia[1:3]
            print(subfamilia)
Ejercicio de agregar una lista frutas y si ya existe una determinada fruta eliminarla despues ver el resultado y recorrer la lista con un ciclo for 
            listFrutas=["uva","pera","manzana","banana"]
            print(listFrutas)
            listFrutas.append("naranja")
            print(listFrutas)

            #condicinarl
            if "pera" in listFrutas:
                listFrutas.remove("pera")

            print(listFrutas)
            print("--ciclo for ----")
            for fruta in listFrutas:
                print(fruta)
